API reference

Import: {{ "import { BadgeComponent } from 'ngx-twcss';" }}

Selector: tw-badge, [tw-badge]

Interface: Badge

Properties Description
{{'@Input()'}} class: string
The style of the badge.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideBadgeConfig() function
  • - Component instance using setConfig method
setConfig(config: Partial<BadgeConfig>): void setConfig(
config: Partial<BadgeConfig>
): void
