API reference for Combobox

Import: {{ "import { ComboboxModule } from 'ngx-twcss';" }}

Selector: tw-combobox

Interface: Combobox

Properties Description
{{'@Input()'}} label: string

The combobox label.

{{'@Input()'}} value: string

The combobox input value.

{{'@Input()'}} opened: boolean

The state of the combobox. Default is false

{{'@Input()'}} inputClass: string

The combobox input class names. e.g. 'ring-2 ring-inset'

{{'@Input()'}} inputMinLength: number

The minimum length of the input value to trigger the filter.
Default is 2

{{'@Input()'}} width:
'w-64' | 'w-72' | 'w-80' | 'w-96'

The combobox width. Default is 'w-72'

{{'@Output()'}} onToggle:
EventEmitter<boolean>

Event emitted when the combobox is toggled.
Emits the state of the combobox.

{{'@Output()'}} onChange:
EventEmitter<string>

Event emitted when the combobox input value changes.
Only emits the typed value. Don't triggered on select().

{{'@Output()'}} onReset:
EventEmitter<void>

Event emitted when the reset icon is clicked.

open(): void

Opens the combobox.

close(): void

Closes the combobox.

select(item: ComboboxItem): void select(item: ComboboxItem):
void

Selects the combobox item.

isSelected(item: ComboboxItem): boolean isSelected(item: ComboboxItem):
boolean

Checks if the combobox item is selected.

scrollIntoView(item: ComboboxItem): void scrollIntoView(item: ComboboxItem):
void

Scrolls the selected combobox item into view.

API reference for Combobox Item

Import: {{ "import { ComboboxItemComponent } from 'ngx-twcss';" }}

Selector: tw-combobox-item

Interface: ComboboxItem

Properties Description
{{'@Input()'}} id: string

The combobox item id.
Default is uuid()

{{'@Input()'}} value: string

The combobox item value.

{{'@Input()'}} selected: boolean

The state of the combobox item.
Default is false

{{'@Input()'}} combobox: Combobox

The combobox reference.

{{'@Output()'}} onSelect:
EventEmitter<ComboboxItem>

Event emitted when the combobox item is selected.

select(): void

Selects the combobox item on click.
