API reference

Import: {{ "import { DropdownComponent } from 'ngx-twcss';" }}

Selector: tw-dropdown

Interface: Dropdown

Properties Description
{{'@Input()'}} class: string
The style of the dropdown.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideDropdownConfig() function
  • - Component instance using setConfig method
{{'@Input()'}} opened: boolean

The state of the dropdown.

toggle(): void

Toggles the state of the dropdown.

open(): void

Open the dropdown.

close(): void

Close the dropdown.

setConfig(config: Partial<DropdownConfig>): void setConfig(
config: Partial<DropdownConfig>
): void

Set the dropdown style configuration.

{{'@Output(\'toggle\')'}} onToggle: EventEmitter<boolean> {{'@Output(\'toggle\')'}} onToggle:
EventEmitter<boolean>

Event emitted when the dropdown state changes.
