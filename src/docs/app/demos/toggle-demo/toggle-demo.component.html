API reference

Import: {{ "import { ToggleComponent } from 'ngx-twcss';" }}

Selector: tw-toggle

Interface: Toggle

Properties Description
{{'@Input()'}} class: string
The style of the toggle.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideToggleConfig() function
  • - Component instance using setConfig method
{{'@Input()'}} slider: string
The style of the slider.
Can be customized through:
  • - HTML template using [slider] attribute
  • - DI using provideToggleConfig() function
  • - Component instance using setConfig method
{{'@Input()'}} checked: boolean

The state of the toggle.

{{'@Input()'}} match:
'first' | 'last' | 'exact'

The matching strategy to be use to resolve the class name.
Default is 'exact'

toggle(): void

Toggles the state of the toggle component.

{{'@Output(\'toggle\')'}} onToggle:
EventEmitter<boolean>

Event emitted when the toggle state changes.
