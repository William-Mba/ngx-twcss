Tonal buttons

A tonal button is an alternative middle ground between primary and secondary buttons. They’re useful in contexts where a lower-priority button requires slightly more emphasis than a secondary would give, such as "Next" in an onboarding flow.

API reference

Import: {{ "import { ButtonComponent } from 'ngx-twcss';" }}

Selector: tw-button

Interface: Button

Properties Description
{{'@Input()'}} class: string
The style of the button.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideButtonConfig() function
  • - Component instance using setConfig method
{{'@Input()'}} size:
'xs' | 'sm' | 'md' | 'lg' | 'xl'

The size of the button.
Default is 'md'

{{'@Input()'}} fab: boolean

Whether the button is a Floating Action Button (FAB).
Default is false

{{'@Input()'}} icon: boolean

Whether the button is an icon button.
Default is false

{{'@Input()'}} variant: ButtonVariant

The variant of the button.
Default is 'primary'

{{'@Input()'}} match: 'first' | 'last'

The strategy to use to resolve the class names.
If 'first', the first part of the class name is used as prefix otherwise the last part is used.
Default is 'first'

setConfig(config: Partial<ButtonConfig>): void setConfig(
config: Partial<ButtonConfig>
): void

Set the button style configuration.
