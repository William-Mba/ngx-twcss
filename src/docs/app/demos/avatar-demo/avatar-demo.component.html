API reference

Import: {{ "import { AvatarDirective } from 'ngx-twcss';" }}

Selector: tw-avatar, [tw-avatar]

Interface: Avatar

Properties Description
{{'@Input()'}} class: string
The style of the avatar.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideAvatarConfig() function
  • - Component instance using setConfig method
{{'@Input()'}} size:
'xs' | 'sm' | 'md' | 'lg' | 'xl'

The size of the avatar.
Default is 'md'

setConfig(config: Partial<AvatarConfig>): void setConfig(
config: Partial<AvatarConfig>
): void

Set the avatar style configuration.
