API reference

Import: {{ "import { ButtonGroupComponent } from 'ngx-twcss';" }}

Selector: tw-button-group, [tw-button-group]

Interface: ButtonGroup

Properties Description
{{'@Input()'}} class: string
The style of the button group.
Can be customized through:
  • - HTML template using class attribute
  • - DI using provideButtonGroupConfig() function
  • - Component instance using setConfig method
setConfig(config: Partial<ButtonGroupConfig>): void setConfig(
config: Partial<ButtonGroupConfig>
): void

Set the button group style configuration.
